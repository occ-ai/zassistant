# 中文编程助手

这是一个 VS Code 插件，自动将中文符号转换为英文符号，提升中文编程体验。输入中文符号时会自动替换为对应英文符号，和原生英文符号输入无异。比如：**输入。会原生输入 . 并弹出代码提示。**

## 功能特色
<img width="700" height="400" src="https://occai.oss-cn-beijing.aliyuncs.com/zyy.gif" >

  
- 自动将常见中文符号（如括号、逗号、引号等）转换为英文符号，并保留原有命令执行。
- 支持通过状态栏一键切换功能启用/禁用
- 可配置是否启用空格键接受补全，默认关闭
- 支持所有语言环境，无需额外配置


## 使用方法

- 插件激活后，输入中文符号会自动转换为英文符号。
- 可通过状态栏按钮切换中英文符号自动转换功能。
- 可在设置中配置：
  - `zassistant.enable`：是否启用符号自动转换（默认开启）
  - `zassistant.enableSpaceAccept`：是否启用空格键接受补全（默认关闭）
